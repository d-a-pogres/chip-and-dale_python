# Чип и Дейл спешат на помощь

**Платформер с элементами аркады по мотивам классического диснеевского мультсериала**

## О проекте

Игра "Чип и Дейл спешат на помощь" - это классический 2D-платформер, созданный на Python с использованием библиотеки PyGame. Игрок управляет знаменитыми бурундуками-спасателями Чипом и Дейлом, сражаясь с врагами и проходя уровни.

### Цель игры
Пройти три уровня, победить всех врагов и финального босса - злого кота Котомрыска!

## Особенности

- **Смена персонажей**: Игрок может переключаться между Чипом и Дейлом во время игры
- **Три уникальных уровня**: 
  - Лесной уровень
  - Городской уровень  
  - Лабиринт Котомрыска
- **Разнообразные враги**: Наземные, летающие и босс
- **Система очков и жизней**: Зарабатывайте очки за победу над врагами
- **Меню и пауза**: Полноценная система меню и паузы

## Управление

| Клавиша | Действие |
|---------|----------|
| **← →** | Движение влево/вправо |
| **ПРОБЕЛ** | Прыжок |
| **Z** | Атака (выстрел снарядом) |
| **X** | Смена персонажа |
| **ESC** | Пауза/Выход из меню |
| **R** | Рестарт игры (после проигрыша) |
| **N** | Следующий уровень |

## Игровой процесс

### Уровни
1. **Лесной уровень** - 3 врага
2. **Городской уровень** - 4 врага  
3. **Лабиринт Котомрыска** - 4 врага + босс

### Механики
- **Прыжки и платформы**: Используйте платформы для перемещения по уровням
- **Атака**: Выстреливайте снарядами по врагам
- **Коллизии**: Избегайте столкновений с врагами
- **Смена персонажа**: Используйте сильные стороны каждого бурундука

### Враги
- **Крысы (наземные)**: Ходят туда-сюда по платформам
- **Пчелы (летающие)**: Летают волнообразно
- **Котомрыск (босс)**: Следует за игроком, больше и сильнее

### Ключевые классы
- **`Player`**: Управление персонажем, физика, атака
- **`Enemy`**: Поведение врагов, паттерны движения
- **`Level`**: Конфигурация уровней, платформы, враги
- **`Game`**: Основной игровой цикл, состояния игры
- **`SpriteManager`**: Загрузка и управление спрайтами

## Графика и дизайн

### Цветовая схема
- **Фон**: Темно-синий (#1a1a2e)
- **Чип**: Оранжевый (#f9a826)
- **Дейл**: Красный (#e94560)
- **Платформы**: Синий (#0f3460) с красной окантовкой
- **Враги**: Коричневый, розовый, красный

### Спрайты
Игра поддерживает как загруженные спрайты, так и автоматически создаваемые fallback-версии при отсутствии файлов.
